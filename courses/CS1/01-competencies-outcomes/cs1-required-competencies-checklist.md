# CS1 Required Competencies Checklist

CS1 focuses on foundational web development and introductory full-stack integration.  
Students are expected to build structured webpages, create responsive layouts, add interactivity with JavaScript, and connect a Next.js frontend with a FastAPI backend through a book project with CRUD operations.

## 0. GitHub Desktop (minimum workflow)

- [ ] Can clone a repository using GitHub Desktop
- [ ] Can make changes to a local file
- [ ] Can create a commit with a clear message
- [ ] Can push changes to GitHub
- [ ] Can pull updates from GitHub
- [ ] Can create and switch to a branch
- [ ] Can open a pull request

### Performance Task

- [ ] Create a personal repository and publish one update using GitHub Desktop
- [ ] Complete one branch-based contribution and open a pull request

---

## 1. HTML / CSS / Bootstrap

- [ ] Can build a basic webpage structure using HTML
- [ ] Can use CSS to style text, spacing, color, and layout
- [ ] Can use Bootstrap components and utility classes appropriately
- [ ] Can create a clear and functional login page
- [ ] Login page includes a heading/title
- [ ] Login page includes a username or email input
- [ ] Login page includes a password input
- [ ] Login page includes a login button
- [ ] Page layout is readable and visually organized

---

## 2. Layout: Bootstrap Grid and Flexbox

- [ ] Can use the Bootstrap 12-column grid system for at least **one** page section
- [ ] Can place content into rows and columns correctly in that section
- [ ] Can use Flexbox for horizontal alignment in at least **one** page section
- [ ] Can use Flexbox for vertical alignment, spacing, and centering where appropriate
- [ ] Can choose Grid vs Flexbox appropriately for each section’s layout needs

---

## 3. JavaScript DOM Manipulation

- [ ] Can select DOM elements using JavaScript
- [ ] Can respond to button clicks and other user actions
- [ ] Can read input values from a form
- [ ] Can update text or content dynamically
- [ ] Can change styles or classes dynamically based on state (optional but encouraged)
- [ ] Can show or hide content (e.g. error message, help text)
- [ ] Can add interactivity to a **form-based page** (e.g. interactive login page or interactive content section), including simple validation

---

## 4. Next.js

- [ ] Can create a basic Next.js project
- [ ] Can understand and use page-based routing
- [ ] Can create multiple pages
- [ ] Can organize UI using reusable components
- [ ] Can pass data through props
- [ ] Can render data on the page
- [ ] Can build and use forms in the frontend
- [ ] Can fetch data from an API
- [ ] Can display backend data on the frontend
- [ ] Can submit frontend data to the backend

---

## 5. FastAPI

- [ ] Can create a basic FastAPI project
- [ ] Can create API endpoints
- [ ] Can return JSON responses correctly
- [ ] Can implement Create in the book project
- [ ] Can implement Read in the book project
- [ ] Can implement Update in the book project
- [ ] Can implement Delete in the book project
- [ ] Can test API endpoints successfully
- [ ] Can explain the purpose of each CRUD operation

---

## 6. Final Course Outcome

By the end of CS1, students should be able to build a simple full-stack web application that includes:

- [ ] A structured frontend page
- [ ] Responsive layout using Bootstrap Grid and Flexbox appropriately
- [ ] JavaScript-based interactivity
- [ ] A Next.js frontend
- [ ] A FastAPI backend
- [ ] CRUD functionality in a book project
- [ ] Successful frontend-backend connection
